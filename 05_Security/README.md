# 🔐 CCNA Security Labs

This section contains all Security-related networking labs completed in Cisco Packet Tracer.  
These labs focus on network protection, access control, and basic security configuration.

---

## 📌 Labs Included

### 1. Basic ACL (Access Control List)
- Configure standard ACL
- Control traffic between networks
- Permit/deny specific hosts or networks

### 2. Extended ACL
- Control traffic based on source and destination IP
- Apply ACL on interfaces (inbound/outbound)
- More granular security control

### 3. Named ACL
- Use named access lists instead of numbers
- Easier management and readability

- Network access restriction techniques
- Real-world network security implementation
