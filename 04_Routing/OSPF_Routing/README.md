# OSPF Routing Lab

## Objective
Configure OSPF between two routers and enable communication between different networks.

## Topology
PC1 → R1 → R2 → PC2

## Networks
- 192.168.1.0/24 (LAN 1)
- 192.168.2.0/24 (LAN 2)
- 10.0.0.0/30 (R1-R2 link)

## Protocol
OSPF Area 0

## Result
Ping successful between PC1 and PC2 using OSPF routing.
