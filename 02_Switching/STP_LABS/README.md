# STP Lab (Spanning Tree Protocol)

## Objective
To understand how STP prevents loops in a Layer 2 network.

## Topology
3 Switches connected in triangle.

## Key Concepts
- Root Bridge Election
- Port Roles (Root, Designated, Blocking)
- Loop Prevention

## Commands Used
- show spanning-tree
- spanning-tree vlan 1 root primary

## Result
STP blocked one redundant link to prevent loop.
