# PortFast + BPDU Guard Lab

## Objective
To configure PortFast for fast access port forwarding and secure it using BPDU Guard.

## Key Concepts
- PortFast: Immediate forwarding for end devices
- BPDU Guard: Protects against unauthorized switches

## Commands
- spanning-tree portfast
- spanning-tree bpduguard enable

## Result
PortFast improved speed, BPDU Guard protected network by disabling unsafe ports.
